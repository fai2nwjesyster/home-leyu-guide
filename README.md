# home-leyu-guide

## 项目简介

`home-leyu-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面资源的集合与索引，方便管理和访问各类静态页面。

## 目录结构

```
home-leyu-guide/
├── index.html          # 主入口页面
├── pages/              # 独立 HTML 页面存放目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库的首页，可作为页面导航或简介。
- `pages/`：存放所有独立 HTML 页面，每个文件可单独访问。
- `assets/`：存放页面共用的静态资源，如样式表、脚本和图片。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档文件，彼此之间无直接关联。每个页面可视为一个独立的文档或工具页面，适用于需要单独展示或引用的场景。

- 页面内容不涉及具体网站、域名或商业服务。
- 页面可包含静态展示、简单交互或信息呈现。
- 所有页面均使用标准 HTML/CSS/JavaScript 编写，无需后端支持。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/home-leyu-guide.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 若需部署到静态托管服务，可直接上传仓库内容。

## 维护说明

- 新增页面请放入 `pages/` 目录，并保持命名规范（如无特殊要求，使用英文小写和连字符）。
- 如需更新公共资源，请修改 `assets/` 下的对应文件，并确保不影响已有页面。
- 不建议在页面中引入外部不可控资源，以免影响可用性。
- 本仓库不负责页面内容的审核或更新，仅提供归档和索引功能。
